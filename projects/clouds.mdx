### Description

Render of moving clouds on a blue sky. Rendered with WebGL and procedurally generated using value noise and fractal Brownian motion.

<CenteredImage src="../clouds/cloud.gif" alt="Cloud animation with shaders" />
<br />

### Inspiration

This is the sequel to the cloud animation in the <InternalLink href="./pixelart">pixelart</InternalLink> project.

<CenteredImage
  src="../pixelart/cloud.gif"
  alt="Cloud animation in pixelart"

/>

Equipped with knowledge of GLSL from [CS4247 Graphics Rendering Techniques](https://nusmods.com/courses/CS4247/graphics-rendering-techniques) and HLSL from my internship at [teamLab](teamlab.art), I wanted to render the moving clouds using WebGL for a long-awaited sequel to the cloud animation in the pixelart project.
